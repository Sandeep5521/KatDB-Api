TheKatDB
katDB is a open online database of Movies and TV Shows with download links. There are many other sites online but none offered a API which is useful when you have find and download a Movie or a Series.This API is fully based on the KatMovieHD website which is widely used for downloading the Movies and Tv Shows.
API Methods
Search movie by name
https://katdb-bogk.onrender.com/movies?name=dam999
Search movie by id
https://katdb-bogk.onrender.com/movies?id=6408569a6585fe8cb3739a68
Search movies by tag
https://katdb-bogk.onrender.com/movies?tag=action
Search movies by year
https://katdb-bogk.onrender.com/movies?year=2023
Get a random movie
https://katdb-bogk.onrender.com/random?type=movie
Get a random show
https://katdb-bogk.onrender.com/random?type=show
Search show by name
https://katdb-bogk.onrender.com/shows?name=showName
Search shows by tag
https://katdb-bogk.onrender.com/shows?tag=drama
Search show by id
https://katdb-bogk.onrender.com/shows?id=64058a95954464bde3af0039
Get list of movies by page number
https://katdb-bogk.onrender.com/movies?page=1
Get list of showss by page number
https://katdb-bogk.onrender.com/shows?page=1
